Política de Privacidade Última atualização: 29 de junho de 2026 Social Analytics para ONGs O aplicativo Social Analytics para ONGs foi desenvolvido como um Projeto de Extensão Curricularizada com o objetivo de auxiliar organizações sem fins lucrativos na análise de métricas públicas de suas redes sociais. Informações coletadas O aplicativo acessa somente informações autorizadas pela Meta Graph API, incluindo: Número de seguidores; Quantidade de publicações; Curtidas; Comentários; Métricas de engajamento. Nenhuma informação pessoal dos usuários finais é coletada, armazenada ou compartilhada. Uso das informações Os dados são utilizados exclusivamente para: geração de dashboards; análise de desempenho das redes sociais; acompanhamento do crescimento das páginas administradas. Compartilhamento de dados O aplicativo não vende, compartilha ou comercializa quaisquer informações. Segurança Os tokens de acesso são utilizados apenas para autenticação junto à Meta Graph API e não são disponibilizados publicamente.
